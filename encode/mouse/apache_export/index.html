ENCODE RNA Dashboard (mm9)

This page provides a summary of transcriptome data production within the ENCODE project.

 

Please observe the guidelines of the "ENCODE Consortium Data Release Policy Summary" when using ENCODE data.

 

The number of experiments produced for each combination of cell line + compartment + RNA fraction + technology is reported in the colored cells.

Clicking on a cell will expand its content, which comprises links to the ENCODE DCC and to various processed data, when available.

 

Full list of data files & batch processing

The full list of data files is available in this index file (see the parsing/download instructions), which should be used for batch file processing.

 

Documentation

When available, README files for each view are made available in this repository. The file naming format of the README files is: "README_{view}.{extension}"
